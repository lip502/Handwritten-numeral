# MATLAB手写数字识别

#### 介绍
该课题为基于MATLAB的特征匹配的数字识别。带GUI界面，可识别0-9十个阿拉伯数字，GUI做成具有滚屏的效果，每识别完一个数字，该数字就滚到旁边显示。可二次开发成语音九宫格的数字识别系统。带论文，
